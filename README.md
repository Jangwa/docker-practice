# How to create a Docker Container?
## What is Docker? -> A box to run your application(almost any application) 
without any 

- OS dependancy error
- Library version mismatch error

This box is called a container, and it 
will include everything that required by your application to run successfuly.
Many time when you find difficulty during installing development environment
in developpers machine - THINK OF DOCKER, Docker Containers will solve your problem
